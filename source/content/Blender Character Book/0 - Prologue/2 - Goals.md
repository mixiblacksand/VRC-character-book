
Our goal here is to create a bipedal character in Blender, bring it into Unity, and make it a functional avatar in VRChat.

We will be using only free tools. No Substance Painter, no fancy paid Blender plugins, not even a tablet.

While most of what we use will be free open source software like Blender and Krita, I will be working on Windows. If you have a working VR installation in Linux, you should be able to follow along with this guide fine.

We'll learn a variety of techniques, but the bulk of the work will be done in a 'sculpt and retopologize' fashion. That is, we'll first create the overall shape of the character we want without regard for polygon count or perfect topology, and then refine it into a game ready model.

Understand that this guide represents my opinions and biases based on being a hobby 3D artist for years. There are a multitude of ways to tackle any problem in 3D art. I'm going to intentionally limit the number of them I present in order to make the guide more digestible for a first time learner. This is only meant to be a stepping stone.
