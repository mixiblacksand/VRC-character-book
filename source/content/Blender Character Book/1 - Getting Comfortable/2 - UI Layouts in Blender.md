By default, Blender should open to this scene:

![[Pasted image 20260331105126.png]]

Blender's UI is divided up into adjustable areas. The main area in the default layout is the 3D Viewport. At the bottom is the Timeline. At the top right is the Outliner. At the bottom right is the Properties.

If you hover your mouse over the border of any two areas, you can click and drag to resize them. Right clicking on the border will bring up a menu allowing you to split or join areas.

![[Pasted image 20260331105512.png]]

In the upper left of each area is an icon that opens a menu of area contents. There's quite a lot here, but we shouldn't be switching these often.

![[Pasted image 20260331105712.png]]

At the top of the screen next to the menus are a number of layout tabs and a plus button. If we come up with a layout we find useful, we can save it here by clicking the tab and choosing 'Duplicate Current.'

![[Pasted image 20260331105950.png]]

Layouts are saved in scene files. This can sometimes be confusing if you load another person's file and their layout is very different to what you're used to. To prevent the layout from being loaded, when opening a file click the gear icon in the upper right, then uncheck 'Load UI.'

![[Pasted image 20260331110247.png]]

The scene you see when you load Blender is saved in a special default scene file. If you'd like to default to something different - say, without the default objects in the scene, make that change and choose File > Defaults > Save Startup File.

![[Pasted image 20260331110424.png]]
