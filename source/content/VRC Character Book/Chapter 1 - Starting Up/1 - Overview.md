# Goal
This is document is designed to be a textbook for people trying to learn a little bit of Blender and Unity in order to make VRChat avatars. We'll focus on a subset of skills and features that address that goal.

I hope that this hybrid book/video format helps some folks that have struggled with the existing video tutorial environment. I'll try to keep segments brief so you can take frequent breaks.
# What You'll Need
We'll be using the following software:
	Blender, our modeling and texturing software
		https://www.blender.org/
	Krita, to assist us in texturing
		https://krita.org/en/
	ALCOM, a FOSS alternative to VRChat Creator Companion
		https://vrc-get.anatawa12.com/en/alcom/
	Unity, which you will install in the process of installing ALCOM.
	VRChat with an account that has upload permissions.

While I'll be using Windows 11, this selection of software should work on Linux as well.

You should have some kind of idea for a bipedal character. We don't want to get too complicated for a beginner project. The key bits are having two legs, two arms, five or fewer fingers, and mostly-human-length neck. Everything else can be pretty easily fudged, but we need to abide by some Unity limitations to keep things simple for now.
# What You Don't Need
You don't need a detailed reference sheet. I'll go over how to use reference, but you can doodle a character entirely in 3D.

A tablet. They're great and you can use one if you want, but I'm not going to assume everyone has one.

A numpad. Blender can make great use of the numpad, but unless you really like traditional roguelikes, spreadsheets, or 3D art programs odds are you bought a keyboard without one these days.
# I'm Opinionated
A lot of us are. Everything you read in this book is biased by my quirks and experiences doing this for both fun and money for years. That's why we need a thousand Blender tutorials - everyone teaches differently and everyone learns differently. If you think this book is trash, it doesn't mean you can't learn Blender. It probably means you just need a different teacher.

I also encourage you to mess around as you go, playing with options and tools I might gloss over. You might find something that suits you perfectly. Just save frequently!
